# APT Falconer Showcase Notes

APT Falconer can showcase this hunt as an investigation workflow:

1. start from the detection;
2. pivot to process ancestry;
3. expand descendants;
4. correlate file writes;
5. pivot to destination infrastructure;
6. record benign look-alikes;
7. produce response notes.

APT Falconer is not required to complete the hunt.

The value being demonstrated is the investigation sequence, not product dependency.
